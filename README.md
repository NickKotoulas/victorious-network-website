# Victorious Network Website

Premium AI-coded marketing website foundation for Victorious Network.

This project is intentionally minimal for now. It provides the technical base for a future bilingual EN/GR AI-driven marketing and communications agency website.

## Stack

- Next.js with the App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- Vercel-ready project structure

## Getting Started

Install dependencies:

```bash
npm install
```

Run the local development server:

```bash
npm run dev
```

Open `http://localhost:3000` in your browser.

## Useful Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
npm run typecheck
```

## Project Structure

```text
src/
  app/          App Router routes, layout, metadata, and global styles
  assets/       Future local media and brand assets
  components/   Reusable UI and layout components
  content/      Future EN/GR content, copy, and structured marketing data
  lib/          Shared utilities and configuration helpers
  sections/     Page sections for future marketing pages
  styles/       Shared style tokens or CSS modules if needed
```
