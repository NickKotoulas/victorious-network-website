# Victorious Network Website

Premium AI-coded marketing website for Victorious Network.

The first English homepage is now in place for an AI-driven marketing and communications agency focused on technology, robotics and innovation-driven brands. The structure remains prepared for future Greek content and deeper site pages.

## Stack

- Next.js with the App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- Vercel-ready project structure

## Getting Started

Install dependencies:

```bash
npm install
```

Run the local development server:

```bash
npm run dev
```

Open `http://localhost:3000` in your browser.

## Useful Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
npm run typecheck
```

## Project Structure

```text
src/
  app/          App Router routes, layout, metadata, and global styles
  assets/       Future local media and brand assets
  components/   Reusable UI and layout components
  content/      Homepage copy and future EN/GR structured content
  lib/          Shared utilities and configuration helpers
  sections/     Homepage sections and future marketing page sections
  styles/       Shared style tokens or CSS modules if needed
```

## Notes

- The current build includes only the homepage.
- Links use placeholder hash targets or future-ready anchors.
- Full bilingual routing, final brand assets, inner pages and production copy can be added later.
